JSON 자료 활용하기

자료 전달방식: XML, JSON

        JSON: Javascript Object Notation
        ==> 모바일(웹)에서의 서버와 데이터 전송방식에서 사용
        특징: 프로그램의 종류나 플랫폼에 관계없이 사용 가능하다.(예: Java, C, C++, C#, PHP, 코틀린, Go언어...등등)
            처리 방식은 Objcet를 String 형식으로 직렬화(serialize) 처리를 해서 전송한다.
            따라서 다시 사용하고자 한다면 JSON 문서형식을 Javascript Object 형식으로 변환하여 사용해야 한다.
        1. JSON 데이터 형식: "key":"value"      <<== JSON에서의 자료는 꼭 큰따옴표로 표시한다!
        2. JSON 객체(중괄호로 둘러싼다): {"key":"value"}
        3. JSON 배열(대괄호로 둘러싼다):
            {
                "회원정보" : [
                    {"성":"홍", "이름":"길동"},
                    {"성":"김", "이름":"말숙"},
                    {"성":"이", "이름":"기자"},
                    {"성":"박", "이름":"세리"}
                ]
            }
        4. JSON 텍스트를 Javascript 객체로 변환: 변환방식 ==> stringify(), parse()
            (1) JSON.stringify(): 자바스크립트 객체를 JSON 형식의 문자열로 변경
            (2) JSON.parse(): JSON 형식의 문자열을 Object(자바스크립트 객체)형식으로 변경
